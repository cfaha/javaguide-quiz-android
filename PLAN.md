# JavaGuide Quiz Android - Full Plan

## Goal
Build an Android quiz app based on JavaGuide content with single-choice, multiple-choice, and fill-in-the-blank questions.

## Milestones
1. MVP (Weeks 1-4): architecture, question model, quiz flow, wrong-book, favorites.
2. Enhancement (Weeks 5-8): mock exam, search, sync metadata.
3. Release (Weeks 9-12): content expansion, polish, release workflow.

## Architecture
- Kotlin + Jetpack Compose
- MVVM + Clean Architecture
- Room for offline-first local persistence

## Core Features
- Category practice
- Random practice
- Wrong-book practice
- Instant scoring + explanation
- Learning progress dashboard

## Quality Gates
- Core flow test pass
- Crash-free > 99.5%
- CI builds APK on each push

## Delivery
- GitHub repository with CI
- APK artifact from GitHub Actions
